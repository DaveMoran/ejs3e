Looping a Triangle

Write a loop that makes seven calls to console.log to output the following triangle:

          #
          ##
          ###
          ####
          #####
          ######
          #######

Answer


          let triangleString = "";
          for(let x = 0; x < 7; x++) {
            triangleString += "#";
            console.log(triangleString);
          }
